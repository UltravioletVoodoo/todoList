# todoList
A simple todo list implemented in python. The goal of this project was to become more familiar with regular expressions

##### Dependencies
- Python 3

##### Running the project
`python todo.py`

##### Sample commands
```
[HELPER]> print
[HELPER]> add string
[HELPER]> add string index
[HELPER]> pop
[HELPER]> pop index
[HELPER]> clear/cls
[HELPER]> exit/q
```
